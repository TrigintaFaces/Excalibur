# Enhanced Stores Example

Demonstrates advanced event store and persistence configurations using Excalibur's enhanced store features.

## Projects

| Project | Description |
|---------|-------------|
| [ECommerceSample](ECommerceSample/) | E-commerce domain with enhanced event stores, snapshot strategies, and projection pipelines |

## What You'll Learn

- Configuring enhanced event store features (compression, encryption, partitioning)
- Snapshot strategies for aggregate hydration optimization
- Building materialized view projections from event streams
