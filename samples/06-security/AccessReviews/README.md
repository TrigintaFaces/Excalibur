# Access Reviews Sample

Demonstrates the A3 Governance access review lifecycle: creating review campaigns, recording approve/revoke decisions, and observing expiry policy behavior using in-memory stores.

## What You'll Learn

- Configuring `AddGovernance()` with access review options
- Creating access review campaigns with default duration and expiry policies
- Recording review decisions (approve, revoke)
- Handling unreviewed grants via `AccessReviewExpiryPolicy`

## Run

```bash
dotnet run
```
