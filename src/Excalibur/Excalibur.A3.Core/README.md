# Excalibur.A3.Core

Lightweight A3 authorization core: in-memory stores, grant management, and authorization evaluation without the full Excalibur stack.

Use `AddExcaliburA3Core()` for standalone scenarios. For the full-stack experience with CQRS, Dispatch middleware, and authentication services, use `Excalibur.A3` with `AddExcaliburA3()` instead.
