# Excalibur.Testing

Testing utilities for Excalibur event-sourced aggregates. Provides fluent Given-When-Then aggregate testing and conformance test kits for IEventStore and ISnapshotStore implementations.

## Installation

```bash
dotnet add package Excalibur.Testing
```

## Quick Start

```csharp
// Add Excalibur.Testing to your service configuration
services.AddTesting();
```
