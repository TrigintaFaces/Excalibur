# Excalibur.Security

Security implementations for Excalibur framework including Argon2id password hashing

## Installation

```bash
dotnet add package Excalibur.Security
```

## Quick Start

```csharp
// Add Excalibur.Security to your service configuration
services.AddSecurity();
```
