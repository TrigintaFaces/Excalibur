# Excalibur.Hosting.Serverless

Core abstractions and interfaces for serverless hosting integration in the Excalibur framework

## Installation

```bash
dotnet add package Excalibur.Hosting.Serverless
```

## Quick Start

```csharp
// Add Excalibur.Hosting.Serverless to your service configuration
services.AddHostingServerless();
```
