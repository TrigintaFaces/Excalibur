# Excalibur.A3.Abstractions

Abstractions for Authentication, Authorization, and Auditing in the Excalibur framework.

## Installation

```bash
dotnet add package Excalibur.A3.Abstractions
```

## Key Types

- Authentication interfaces
- Authorization policy interfaces
- Audit logging interfaces
