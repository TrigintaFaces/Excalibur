# Excalibur.EventSourcing.InMemory

In-memory event store implementation for Excalibur.EventSourcing. Intended for testing and development scenarios.

## Installation

```bash
dotnet add package Excalibur.EventSourcing.InMemory
```

## Quick Start

```csharp
// Add Excalibur.EventSourcing.InMemory to your service configuration
services.AddEventSourcingInMemory();
```
