# Excalibur.Cdc

Change Data Capture (CDC) abstractions for the Excalibur framework, providing fluent builder configuration for table change tracking.

## Installation

```bash
dotnet add package Excalibur.Cdc
```

## Features

- Fluent builder API for CDC configuration
- Table-level change tracking setup
- Stale position recovery strategies
- Background processing hosted service
- In-memory CDC provider for testing
