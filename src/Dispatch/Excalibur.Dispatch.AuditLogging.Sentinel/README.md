# Excalibur.Dispatch.AuditLogging.Sentinel

Azure Sentinel audit log exporter for the Dispatch compliance framework. Supports Azure Monitor Data Collector API for custom log ingestion.

## Installation

```bash
dotnet add package Excalibur.Dispatch.AuditLogging.Sentinel
```

## Quick Start

```csharp
// Add Excalibur.Dispatch.AuditLogging.Sentinel to your service configuration
services.AddAuditLoggingSentinel();
```
