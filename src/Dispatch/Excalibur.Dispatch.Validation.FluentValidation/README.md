# Excalibur.Dispatch.Validation.FluentValidation

FluentValidation integration for the Dispatch messaging framework. Provides IValidatorResolver implementation using FluentValidation validators.

## Installation

```bash
dotnet add package Excalibur.Dispatch.Validation.FluentValidation
```

## Quick Start

```csharp
// Add Excalibur.Dispatch.Validation.FluentValidation to your service configuration
services.AddValidationFluentValidation();
```
