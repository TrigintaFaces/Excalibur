# Excalibur.Dispatch.Analyzers

Roslyn analyzers for the Excalibur framework.

## Installation

```bash
dotnet add package Excalibur.Dispatch.Analyzers
```

## Features

- Handler implementation validation
- Message type validation
- Middleware configuration checks
- Best practice enforcement
