# Excalibur.Dispatch.AuditLogging.Datadog

Datadog audit log exporter for the Dispatch compliance framework. Supports the Datadog Logs API for custom log ingestion.

## Installation

```bash
dotnet add package Excalibur.Dispatch.AuditLogging.Datadog
```

## Quick Start

```csharp
// Add Excalibur.Dispatch.AuditLogging.Datadog to your service configuration
services.AddAuditLoggingDatadog();
```
