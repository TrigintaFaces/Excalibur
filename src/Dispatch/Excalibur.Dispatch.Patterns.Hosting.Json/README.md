# Excalibur.Dispatch.Patterns.Hosting.Json

JSON serialization support for Dispatch patterns in hosting scenarios.

## Installation

```bash
dotnet add package Excalibur.Dispatch.Patterns.Hosting.Json
```

## Features

- System.Text.Json serialization
- JSON health check endpoints
- JSON API responses
