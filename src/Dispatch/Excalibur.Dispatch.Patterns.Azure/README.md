# Excalibur.Dispatch.Patterns.Azure

Azure-specific pattern implementations for the Dispatch framework.

## Installation

```bash
dotnet add package Excalibur.Dispatch.Patterns.Azure
```

## Features

- Azure Blob Storage claim check
- Azure Table Storage patterns
- Azure Queue patterns
