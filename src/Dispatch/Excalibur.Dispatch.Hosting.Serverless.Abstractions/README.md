# Excalibur.Dispatch.Hosting.Serverless.Abstractions

Base abstractions for serverless hosting implementations in the Excalibur framework.

## Installation

```bash
dotnet add package Excalibur.Dispatch.Hosting.Serverless.Abstractions
```

## Usage

This package provides common interfaces and types used by serverless hosting packages (Lambda, Azure Functions, Google Cloud Functions).
